# HookEuicc

特定のアプリでの eSIM 要件をバイパスし、eSIM アクティベーションコードの取得を可能にします。

eSIM のアクティベーションコードを直接表示しないアプリの場合は、モジュールがコードを自動的にクリップボードにコピーします。

## 更なる機能

### OMAPI をバイパス

ARA (アクセスルールアプリケーション) と ARF (アクセスルールファイル) の制限をバイパスします。

通常は、ARA が適用されていないカードに OMAPI アクセス権限を付与するために使用されます。

> [!NOTE]
> スコープで `com.android.se` を選択して再起動 (またはこちらを実行: `su -c killall com.android.se`) してください。
>
> 問題が発生していない限りは、この機能を有効化しないでください。
